# nonoroazoro25.github.io

Hugo 博客，使用 Stack 主题，通过 GitHub Actions 部署到 GitHub Pages。

## 若站点只显示占位页、没有主题

请在仓库 **Settings → Pages** 中：

1. 打开 **Build and deployment**
2. 将 **Source** 设为 **GitHub Actions**（不要选 “Deploy from a branch”）
3. 保存后到 **Actions** 里运行或重新运行 “Deploy Hugo site to Pages” workflow

部署成功后，访问 https://nonoroazoro25.github.io/ 即可看到完整博客。
